# Mermaid Editor

一个功能丰富的在线 Mermaid 图表编辑器，支持实时预览、可视化配置和多种导出格式。

## 功能特性

- **实时编辑预览** - 边写边看，即时反馈
- **可视化配置** - 通过图形界面调整图表样式，无需修改代码
- **灵活的排版控制** - 节点间距、排列方向、连线样式等可调
- **丰富的主题配置** - 自定义颜色、字体、背景等
- **多格式导出** - 支持 SVG 和 PNG 格式下载
- **交互式视图** - 缩放、平移等操作

## 项目结构

```
├── mermaid_editor.html          # 主编辑器（推荐版本）
├── mermaid_editor_Version2.html # 备选版本
├── flowchart_Version2.html       # 流程图专用编辑器
└── diagram*.svg                  # 示例图表
```

## 快速开始

1. 打开 HTML 文件在浏览器中运行
2. 在左侧编辑面板输入 Mermaid 代码
3. 右侧即时预览图表
4. 使用下方控制面板调整样式
5. 点击导出按钮下载图表

## 支持的图表类型

- 流程图 (Flowchart)
- 状态图、时序图等 Mermaid 支持的所有图表类型

## 技术栈

- HTML5 + CSS3 + JavaScript
- Mermaid 10.9.0 - 图表渲染
- SVG Pan Zoom 3.6.1 - 缩放平移功能

## 使用示例

编辑器预设了一个复杂的多层级流程图示例，展示了各种样式和功能。
